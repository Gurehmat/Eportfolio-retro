PantryCheff

AI-powered pantry + recipe engine that reduces food waste by cooking from what you already have. Built with React, TypeScript, Supabase (Postgres, Auth, Edge Functions), and Gemini for multimodal fridge scanning.

  • Engineered a Set-based recipe matching algorithm with O(P+R) complexity and confidence scoring (exact, partial, fuzzy), covered by targeted unit tests.
  • Built a multimodal AI pipeline: fridge photo → Supabase Edge Function → Gemini Vision → validated ingredient list added to a type-safe pantry store.
  • Implemented strict TypeScript + Zustand state management, with 154 tests (Vitest, React Testing Library, MSW) running in GitHub Actions CI.
  • Optimized the main bundle by 99.8% (3,572KB → 8KB) with code splitting, lazy loading, and response caching.

AI Lesson Coach

Enterprise AI coaching chatbot built for a real Workplace Edge client as part of a 3-person team. React 18, TypeScript, Express, Prisma, Gemini 2.5 Flash, frontend on Vercel, API on Railway, and Turso (libSQL) as the database.

  • Built a heuristic intent detection pipeline to classify learner requests and dynamically inject lesson content into Gemini's system prompt, adapting responses to the learner's current module.
  • Implemented a content safety layer with PII stripping, keyword-based escalation detection for HR/crisis topics, and input filtering before any data reaches the model.
  • Integrated a SCORM iframe embed so the chatbot renders inside existing LMS course environments without breaking the lesson flow.
